# 四阶段交易法

> **核心理念**：A 股市场存在四浪循环规律，每浪有不同的评分区间、仓位策略、操作纪律。  
> 把市场切成 4 个状态 → 把操作切成 4 套动作 → 用数据替代情绪，用纪律替代直觉。  
> **2026 年 5 月完整执行 4 个阶段（主升浪→顶部反转→高位震荡→杀跌清仓），月度 +4.41% 跑赢上证 +5.47%**。

---

## 一、理论起源：四浪循环理论

> **源文件**：`wiki/SYSTEM_EVOLUTION.md` 第 337-367 行

A 股市场按"时间占比"划分为四浪，每浪有不同的评分区间、仓位策略、操作纪律：

```
阶段1: 主升上涨浪 (20%时间)  ←  赚钱主要靠这一浪
├─ 特点：上涨为主，创出新高
├─ 评分：7.5-9.5 分（持续高位）
├─ 策略：重仓持有（70-80%）
├─ 目标：享受主升浪
└─ 2025-12-24 至 2026-01 中旬处于此阶段

阶段2: 高位震荡浪 (35%时间)
├─ 特点：高位震荡，箱体运行
├─ 评分：4-7 分（上下波动）
├─ 策略：高抛低吸（30-50%）
├─ 目标：区间操作
└─ 2026-01 后半段处于此阶段

阶段3: 调整杀跌浪 (10%时间)  ⚠️ 最关键
├─ 特点：快速杀跌，跌破支撑
├─ 评分：<3 分（极低位）
├─ 策略：空仓观望（0-10%）
├─ 目标：躲避风险，严控回撤
└─ 2026-01-30 后处于此阶段

阶段4: 低位震荡浪 (35%时间)
├─ 特点：低位震荡，筑底准备
├─ 评分：4-7 分（震荡上行）
├─ 策略：逢低吸纳（20-40%）
├─ 目标：等待突破
└─ 杀跌结束后进入
```

### 关键洞察

- **20% 的时间创造绝大部分收益**（主升上涨浪）—— 耐心等待是核心
- **10% 的时间摧毁绝大部分收益**（调整杀跌浪）—— 严控回撤是核心
- **70% 的时间在震荡**（高位+低位震荡浪）—— 不重仓、不预测、控期望

## 二、精炼四阶段操作法

> **源文件**：`wiki/A股投资真谛：基于量化数据的实战规律总结.md` 第 66-83 行（1.2 节）

把四浪循环进一步精炼成"**4 个操作动作**"，每一步都有严格的"评分触发条件"：

### 布局阶段：评分突破 7.5 分，建仓试探
- 首次突破 7.5 分时，小仓位试探性建仓
- 观察后续 2-3 天的表现确认趋势延续性
- **底线**：单次建仓不超过总资金 1/3

### 持仓阶段：评分持续 7.5 分以上，坚定持有不动摇
- **最重要的阶段**，绝对不能被短期震荡吓跑
- 历史数据证明：**评分 ≥ 7.5 分时，上涨概率超过 80%**
- 关键纪律：忽略单日波动，看 3 日均线

### 观察阶段：单日跌破 7.5 分，观察次日修复能力
- 如果次日迅速修复至 7.5 分以上，继续持仓
- **修复力度越大（如 3.6 分的大修复），后续行情越强劲**

### 离场阶段：连续两天跌破 7.5 分，果断清仓离场
- 这是历史验证的**最可靠离场信号**
- 不要心存侥幸，严格执行纪律
- 案例：2026-01-13 评分 7.3 跌破 7.5 → 减仓至 5 成 → 1/14-1/16 三天市场调整

## 三、六子阶段演化（主升浪精修）

> **源文件**：`wiki/SYSTEM_EVOLUTION.md` 第 861-907 行

四浪循环中的"主升上涨浪"内部还有 6 个子阶段，需要更精细的仓位管理：

```
从粗到细的进化：

原系统（四阶段理论）：
└─ 阶段1: 主升上涨浪 (20%)
    ├─ 评分: 7.5-9.5
    ├─ 仓位: 70-80%
    └─ 问题: 一刀切，无法应对内部波动

新系统（六子阶段）：
└─ 阶段1: 主升上涨浪
    ├─ 1.萌芽期: 7.5≤综合<8.8，日线≥8 → 5-7成
    ├─ 2.确认期: 综合≥8.8 → 7-9成
    ├─ 3.初期: 综合≥7.5，日线≥8 → 8-10成
    ├─ 4.中继期: 综合<7.5，日线≥8 → 保持仓位
    ├─ 5.后期: 综合≥7.5，日线≥8 → 8-10成
    └─ 6.尾期: 综合<7.5，日线≥8 → 5-7成
```

### 六子阶段状态机

```
┌─────────────────────────────────────────────────────────────────┐
│                      主升浪六阶段状态机                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   萌芽期 ──────→ 确认期 (综合≥8.8)                             │
│   (5-7成)         (7-9成)                                       │
│      ↓               ↓                                          │
│   [退出]         初期 (综合≥7.5+日线≥8)                         │
│                  (8-10成)                                       │
│                      ↓                                          │
│                  中继期 (综合<7.5+日线≥8)                       │
│                  (保持仓位)                                      │
│                      ↓                                          │
│                  后期 (综合≥7.5+日线≥8)                         │
│                  (8-10成)                                       │
│                      ↓                                          │
│                  尾期 (综合<7.5+日线≥8)                         │
│                  (5-7成)                                        │
│                      ↓                                          │
│                   [退出主升浪]                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**核心创新**：把"中继期"独立出来——主升浪中的回调**不一定要减仓**，只要日线评分 ≥ 8 仍可保持仓位。

## 四、仓位矩阵

| 评分 \ 阶段 | 萌芽期 | 确认期 | 初期 | 中继期 | 后期 | 尾期 |
|------------|--------|--------|------|--------|------|------|
| 综合 ≥ 8.8 | 7-9成 | 7-9成 | 8-10成 | 保持 | 8-10成 | 5-7成 |
| 7.5 ≤ 综合 < 8.8 | 5-7成 | 等待 | 8-10成 | 保持 | 8-10成 | 5-7成 |
| 综合 < 7.5 | 0成 | 0成 | 0成 | 等待 | 0成 | 0成 |

**横向看**：评分越高，仓位越重（顶到 10 成）  
**纵向看**：阶段越往后，越接近"中继→离场"  
**特例**：中继期（综合 < 7.5 但日线 ≥ 8）不主动加减仓

## 七、失败案例与边界条件

四阶段不是万能的，**承认失败才能让系统进化**。

### 失败案例 1：2026-03 逆系统融资加仓（月度 -9.26%）

- **背景**：3 月上证从 4080 跌到 3957，跌破 1 月跳空缺口
- **错误**：阶段 3 杀跌浪中**应该空仓观望**，但逆系统融资加仓
- **结果**：
  - 阶段最大回撤 **6.21%**
  - **月度收益 -9.26%**（是系统运行以来最大单月亏损）
  - 由 2 月底年累计 +6.74% 跌至 3 月底年累计约 -2.5%
- **教训**：3 月下旬清仓优化系统，从 V3 升级到 V6.4，**信任系统、相信系统、相信系统**
- **4/15 AI 评**："从 3 月份逆系统操作导致大幅亏损，到 4 月份集中精力优化系统、连续 4 天正确决策，这段蜕变令人印象深刻。您没有在亏损后消沉，而是将痛苦转化为行动。"

> 源文件：[2026-04-15 自评](trading/2026/2026-04/daily/2026-04-15自评.md) + [2026-04-15 AI 评](trading/2026/2026-04/daily/2026-04-15AI评.md)

### 失败案例 2：2026-02-02 逆系统加仓

- 2 月 9 日预判跳空缺口回补，但 2/2 提前加仓
- 单日亏损 -7,119 元（来自 4/16 自评）
- 教训：**预判只是参考，执行必须等信号**

### 失败案例 3：5/14 V7 策略完整执行的回撤

- 阶段 3 杀跌浪中坚持 V7 "主升浪不减仓"
- 两天回撤 17,000 元（占 5 月月利润 68%）
- **不是失败而是验证**：4 天高位震荡后确认退出，证明系统有效
- **结论**：需要在顶部衰竭阶段引入微调——"主升浪后期检测短周期衰竭，轻度→8 成，重度→7 成"

## 十、结语

四阶段交易法不是万能公式，**它是一个反人性的纪律框架**：
- **人**想追涨杀跌 → **法**告诉你等待 7.5 分信号
- **人**想在主升浪中段止盈 → **法**告诉你保持至连续 2 天跌破
- **人**想在杀跌浪中抄底 → **法**告诉你空仓观望至 9 分反转

**量化评分 + 四阶段交易法 + 严格纪律 = 散户在 A 股唯一可复制的赚钱系统**。
